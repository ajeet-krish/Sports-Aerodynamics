# The Physics of Play

**Aerodynamics in Summer Sports & FIFA World Cup 2026**

Interactive CFD simulations built with [ΦFlow](https://github.com/tum-pbs/PhiFlow), demonstrating core aerodynamic concepts through sports-inspired examples.

## Modules

1. **The World Cup Free-Kick** — Magnus effect on a spinning soccer ball: asymmetric pressure fields and curved trajectories.
2. **The Peloton & Running Pack** — Wake interactions, vortex shedding, and drafting efficiency between athletes.

## Getting Started

```bash
uv sync
```

Then open `physics_of_play.ipynb` in Jupyter.

## Requirements

- Python 3.12
- [uv](https://docs.astral.sh/uv/) package manager
- See `pyproject.toml` for full dependencies

## Outputs

Generated animations are saved to `assets/`. Run all notebook cells to produce them.
